🆘 よくあるエラーと対処法📖 PDF p.23~25

基本の心得： エラーが出たら、まず《フォント》と《パス》を疑う！

❶ フォント・記号エラー

症状： コマンドが動かない、文字化けする
原因： コピペ時に半角→全角、ハイフン→長音などに変わっている
対処： AIに「このコマンドは正しいですか？」と確認

❷ パスエラー

症状： command not found、No such file or directory
原因： インストール場所とパスが一致していない
対処：
  1. which mecab でインストール場所を確認
  2. echo $PATH でパス設定を確認
  3. findコマンドで辞書の実パスを探す
  4. それでもダメならAIに相談

❸ 権限エラー

症状： Permission denied
原因： 管理者権限が必要な操作
対処： コマンドの前に sudo をつけて実行
それでも解決しない場合：
AIにエラーメッセージのスクリーンショットを送って質問 → Slackで質問
→ 焦らず、一つずつ確認していけば必ず解決できます！
